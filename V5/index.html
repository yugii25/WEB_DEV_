Lists and it's Types

  1. Ordered List
    It contains list items in ordered format and it has type attributes which can have values like i, I, 1, a and A, It also has a start attribute which can start from in between order.
    An example of the Ordered list:
    1. 500gm Sugar
    2. 500ml Water
    3. 5 pinches of salt
    4. Proper Stirring
  2. Unordered Lists
    It is used to define list in an unordered manner and it also has type attribute which has values like circle, disc and square. An example of unordered list :
    • Cherry
    • Banana
    • Kiwi
    • Pineapples
  3. Definition Lists
    These lists are used to define terms and it comes with two tags dt and dd where dt is definition term and dd is definition description and an example of this is the list you're seeing right now.
